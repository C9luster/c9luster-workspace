# 工具与脚本

存放日常开发和学习中使用的工具脚本和实用程序。

## 📚 目录说明

本目录用于存放各类工具脚本，包括但不限于：

- 开发辅助脚本
- 数据处理工具
- 自动化脚本
- 系统管理工具
- 测试工具
- 其他实用工具

## 📝 脚本分类

建议按功能分类组织：

```
tools/
├── dev/              # 开发相关工具
├── data/             # 数据处理脚本
├── automation/      # 自动化脚本
├── system/          # 系统管理工具
└── utils/           # 通用工具函数
```

**Note**: 本目录中的脚本仅供学习和个人使用，使用前请仔细检查。
