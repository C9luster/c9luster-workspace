# 编程语言学习目录

这是一个用于学习多种编程语言的目录，包含 Python、Go、Shell 等语言的学习资料和实践代码。

## 目录结构

- [python/](./python/) - Python 语言学习资料和示例代码
- [go/](./go/) - Go 语言学习资料和示例代码
- [sh/](./sh/) - Shell 脚本学习资料和示例代码

## 学习目标

通过这个目录，你可以：
- 学习不同编程语言的基础语法
- 了解各语言的特性和最佳实践
- 通过实际项目练习提升编程能力
- 对比不同语言的实现方式和设计理念

## 使用说明

每个语言目录下都包含：
- 基础语法示例
- 常用库和工具的使用
- 实际项目案例
- 学习笔记和参考资料
